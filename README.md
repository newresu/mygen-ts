Scaffold a simple Typescript project.

```
npm i mygen-ts -g
```

Scaffold looks like this:

```bash
├── LICENSE  # MIT
├── node_modules # installs dev deps
├── package.json # populated
├── README.md
├── src
├── tsconfig.cjs.json
├── tsconfig.esm.json
├── tsconfig.json
└── vitest.config.ts
```
